Card Background Image

Optional image drawn behind all elements. The colour above is still used as the fallback/base layer.

Card Safe Area Padding

Top-level element positions are measured inside this area. So an element at X 0%, W 100% fills the usable design area, not the raw card edge.

Card Sides

Editable front / back
Edit the front and back as separate card designs. Card size, DPI, safe area, radius and border are shared; background and elements are stored separately for each side.

Project

.tcgproject includes images
Save or load the full card project: front/back templates, elements, layers, styles, fonts, CSV data, images, image library, printable sheet settings, and card settings. Project data is generated only when requested for better editing performance.

Export

PNG output
Export the currently previewed card as a PNG file, or copy the rendered PNG to your clipboard for pasting into another app.

Printable Sheets

A4 / US Letter page layout
Lay cards out on full printable pages. By default this creates duplex-ready pairs: a front page followed immediately by its matching back page. Choose Fronts only if you only want single-sided card fronts.

Data Tags / Batch Cards

CSV columns override tagged fields
Give an element a data tag, then import CSV with matching column names. Example: a badge with tag badge uses the value in the badge column. Batch export can save all CSV rows as separate PNGs or one ZIP file.

Image Library

Upload images once, then reference them from CSV image columns by filename. The image library is saved inside project files.

Border Styles

Shared presets for element borders
Create reusable border presets once, then assign them to elements. Duplicate a style to make a similar variation.

Text Styles

Shared presets for font and outline
Create reusable text presets once, then assign them to titles, text boxes, badges, and stat boxes. Font family, size, weight, colour and outline are shared; alignment remains on each element.

Box Styles

Shared presets for fills and padding
Create reusable box presets once, then assign them to titles, text boxes, images, badges, stat boxes, and groups. Border and text styling remain separate.

Elements

Use the Order number to slot an element into the list. Use Relative positioning to place an item below, above, left, or right of another element.

Live Preview

Use Prev/Next to review CSV rows. Turn on the back preview to check both sides together.
No data rows
Project JSON preview is manual-only for better performance. Use Open Project Data in the Project section when you need to inspect it.

Style Inspector

Selected element properties
Select an element to inspect and edit the shared Border, Text, and Box styles it uses. Use the padlock to prevent accidental edits to a shared style.
