How to use: Use the Canvas menu to set the actual image size, save/load editable DashpadSketch files, and export a PNG, or copy/open the canvas image from the clipboard command. Use Selected Object to change Z layering or delete the current object. Freehand pen: drag to draw. Click-point line: left-click points, right-click to finish. Bézier curve: click a start point, then click an end point; drag the two control handles afterwards. Select/move: click an object, then drag its points or drag the line/body to move the whole object. Delete line segment: click a segment between two points. Press Delete to remove the selected object, or Escape to cancel the current drawing action.
